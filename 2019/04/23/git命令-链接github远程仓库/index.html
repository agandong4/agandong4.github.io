git命令 链接github远程仓库

Posted on 2019-04-23 |

git 命令

常用命令

  • git init 初始化某一个文件夹
  • git status
  • git add/rm
    • git rm -rf r 递归 f 强制
  • git remote add orgin 远程仓库地址 添加远程仓库
  • git push -u orgin master

图解

不常用命令

  • git diff

  • git log —oneline

    • Git log —oneline —graph

  • git commit -m “”

    • git commit —amend —no-edit

  • git reset 1.py

    • git reset —hard HEAD
    • git reset —hard HEAD^ 回到一个版本前
    • Two ^ 回到两个版本之前
    • ~100

  • git reflog

    • git reset —hard versionID

  • git checkout — 1.py commitID

  • git branch 查看分支

    • git branch dev
    • git checkout dev 切换到dev分支
    • git checkout -b dev
    • git branch -d dev
    • git merge —no-ff -m “keep merge info” dev

  • git rebase dev 一个高级命令
