用AARRR模型分析714高炮

2018年5月5日晚，刚执行完飞行任务的空姐小李，在郑州航空港区叫了一辆滴滴赶回市里，但她永远不曾到达目的地。

凶手刘某，案发时在某网贷平台有逾期长达111天的记录。

2018年8月24日，温州姑娘小赵搭乘滴滴去参加同学的生日会，司机开到偏僻山路，先劫财、后劫色，最后将她残忍杀害。

凶手钟某，背负了56笔未偿还的现金贷。

这两起凶案，其实都不是单纯的冲动型犯罪。

两个凶手在犯案前，皆已有走投无路、对生活绝望之感。

而他们共同的特点,均是碰了714高炮

什么是714高炮

“714高炮”指的是一些期限为7天或14天的高利息网络贷款,“高炮”是指其高额的“砍头息”及逾期费用.比如向这些平台贷款1000元,能拿到手里的钱是700元,以各种名义收取的300就是所谓的“砍头息”.但其实有时候第七天就需要还款了,变成了“613”或者“512”了.

714高炮的年化贷款利率是1500%-2500%.要知道,银行的一年期利率也才1.75%.今年315曝光的一个真实案例:一个月从7000借款变成4万,三个月涨到50万.没有比这更赚钱的了

AARRR模型

Acquisiton 获取用户

  • 网站推广 一般通过广告联盟进行投放,用户每成功下载安装一次app ,网站能得到2-5元.因此有很多网站选择接这种平台的广告
  • 贷款超市 比如融360平台推荐过的“酷卡”和“贷上钱”.工作人员每成功放出一笔贷款，他们就会收取一定的费用
  • 微信群/QQ群推广 通过买群.拉人的方式,对群里面的人以“小额无抵押放款”等软文进行引导,驱使用户下载app
  • 短信推送 从黑市上购买的一些有过贷款记录的名单,称之为料

Activation 激活用户

  • 通过“最快3分钟放款”,“小额免息”,“无任何担保”等字眼吸引用户,在用户注册账号后,还需要上传身份证照片,个人真实照片,同时还要求提供通信运营商的验证码,以便得到运营商三个月通话记录;私底下还会收集手机信息(比如mac地址,手机序列号IMEI等),以便后期的风险控制
  • 短信再激活 将收集到的信息转手在交易市场转卖出去,新鲜的料可以卖不少钱.所以注册过app的人一天可以收到20-30次其他app短信

Retention 提高留存度

业内有些行话,“强制上岸”(更换手机号,彻底离开原先圈子),“坦白”(向家人坦白,请求原谅),“爆通讯录(对通讯录联系人进行骚扰或更进一步操作)”,大部分贷款者最后只能选择强制上岸或者坦白.

  • 高额利息 1000借一周,到手700,利息30%,6天后就得还1300.走头无路的人没有办法,病急乱投医,只能寻求其他渠道
  • 短信推送 来自自家app的 或者其他家的短信,是这些急于还款的人唯一的选择.因此不难理解有些人背负100多家网络贷款平台的欠款

Revenue 获取收入

有篇文章说多“贩卖米粉都没这么赚钱”,赚钱的速度堪比印钞;这种行业定位就是被银行信贷,信用卡,正规网贷平台排除在外的那群人.虽然坏账率高,但是依靠优秀的风控与催收,仍然可以获取丰厚的利润

  • 房产税的出台,使得一部分炒房资金,流入了网络贷.有钱的人通过中介找到放贷平台,采用利润对半分的方式.即使对半分,利润也高达500%以上
  • 风控系统 网络上有着一群靠着撸网贷的人,人称“撸口子”,有些是曾在714高炮工作过的人,熟悉其操作流程.另外有一些强制上岸的人员,面对这样的“撸贷大军”,没有强大的风控系统,基本会撸到血本无归.放贷公司讲之前收集到的客户信息(包括基本资料,通讯录资料,其他的信息等)输入风控系统,风控返回一个风险的评估值.常见的风控系统有:
    • 在黑市上买的数据库,自己开发的风控
    • 第三方风险控制平台,每份收取十几元到几十元不等.多数是使用同盾,有盾等数据
